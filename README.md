
# MiniGreenHouse

currently making a greenhouse out of a fish tank


## Installation

just clone change settings and upload it to your arduino micro or leonardo

```bash
  git clone https://github.com/MrLuto/MiniGreenHouse.git
  cd MiniGreenHouse
```
